# Flutter SQLite CRUD Example

Минималистичное и чистое Flutter-приложение, демонстрирующее базовые операции CRUD (Создание, Чтение, Обновление, Удаление) с локальной базой данных **SQLite** и архитектурой **MVC (Model-View-Controller)**.

Сделано для изучения основ работы с локальным хранилищем данных во Flutter.

## Технологии и зависимости

* **Flutter** - UI фреймворк
* **Dart** - Язык программирования
* **sqflite** Работа с SQLite
* **path_provider** - Определение путей к файлам

## Особенности

* **Полный цикл CRUD:** Создание, отображение, редактирование и удаление записей
* **Локальная база данных SQLite:** Использование пакета `sqflite` для надежного хранения данных на устройстве
